# Data Analytics Project

## Overview
This project focuses on end-to-end data analytics workflow using Python, MySQL, and Power BI. The objective of the project was to analyze customer shopping behavior, extract meaningful insights, and present the findings through interactive dashboards and reports.

The project includes:
- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using MySQL
- Dashboard creation in Power BI
- Report and presentation preparation using Gamma

---

## Dataset
The dataset contains customer shopping and purchasing information such as:
- Customer demographics
- Purchase amount
- Product categories
- Review ratings
- Discount usage
- Payment methods
- Shopping behavior trends

The dataset was analyzed to identify customer patterns, revenue insights, and purchasing trends.

## Project Workflow

### 1. Data Loading
- Imported CSV dataset into Python using Pandas
- Verified data structure and column information

### 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Corrected data types
- Handled inconsistent values

### 3. Exploratory Data Analysis (EDA)
Performed analysis on:
- Customer purchase behavior
- Revenue trends
- Product performance
- Discount usage
- Review ratings
- Gender-based spending analysis

### 4. SQL Analysis
Loaded cleaned dataset into MySQL and executed SQL queries to:
- Calculate total revenue
- Analyze customer behavior
- Identify top-rated products
- Measure discount impact
- Generate business insights

### 5. Power BI Dashboard
Built an interactive dashboard including:
- Revenue KPIs
- Product analysis
- Customer segmentation
- Discount trends
- Review rating insights

### 6. Reporting & Presentation
- Created a project report summarizing findings
- Developed a presentation using Gamma for project explanation and storytelling
